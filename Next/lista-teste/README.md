## Getting Started / Início

Siga os passos abaixo para instalar e executar este projeto localmente.

### Requisitos

- Node.js (versão 16 ou superior)
- npm, yarn, pnpm ou bun como gerenciador de pacotes

### Instalação

1. Clone o repositório ou copie os arquivos para a sua máquina.
2. Abra um terminal na pasta do projeto (`Next/lista-teste`).
3. Execute o comando de instalação das dependências usando o gerenciador de sua preferência:

```bash
npm install
# ou
# yarn install
# ou
# pnpm install
# ou
# bun install
```

### Execução em desenvolvimento

Após instalar as dependências, inicie o servidor de desenvolvimento:

```bash
npm run dev
# ou
# yarn dev
# ou
# pnpm dev
# ou
# bun dev
```

Abra [http://localhost:3000](http://localhost:3000) no seu navegador para ver a aplicação em funcionamento. O projeto suporta recarregamento automático ao editar arquivos.

### Testes

O projeto possui testes configurados com [Jest](https://jestjs.io) e [React Testing Library].
Para executar a suíte de testes, rode:

```bash
npm run test
# ou
# yarn test
# ou
# pnpm test
```

Os testes serão executados em modo de linha de comando; se desejar, adicione a flag `--watch` para rodar no modo vigilante durante o desenvolvimento.

### Build e execução em produção

Para criar uma versão otimizada e em seguida executá-la:

```bash
npm run build
npm run start
# ou comandos equivalentes com yarn/pnpm/bun
```

Isso gera uma pasta `.next` com os arquivos compilados e inicia o servidor de produção.
